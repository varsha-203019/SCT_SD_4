#Task Title

E-commerce Product Information Extractor

#Description

This program extracts product names and prices from HTML content. It can also save the extracted data to a CSV file for structured storage and analysis.

#Programming Language

Python

#How It Works

Split HTML into lines.

Extract product names using <h2> tags.

Extract prices using <p>Price: tags.

Print the data in a structured format or save to CSV.

Key Concepts Used

String manipulation

HTML parsing (manual)

Loops and conditionals

CSV file handling

#Output

Displays the product names and prices in tabular format or CSV.
